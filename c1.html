Programming Language - C

Home > C 
    C is a structured programming language which means that there is a defined structure that has to be followed while writing the code.

    In general all the programs in C follow the structure given below:
Documentation section
Link section
Definition section
Global definition section
main() function section beginning using {
   Declaration part
   Execution part
End of the main() section using }
Sub program section having user defined functions
function1() {
   - - - - -
   - - - - -
}

    The documentation section consists of a set of comment lines giving the name of the program, the author and other details.
    The comment lines are specified in between /* - - - -- */ which indicates that this part is not an executable part. Single comment lines are 
specified in between //.
    The link section provides instructions to the compiler to link functions from the system library. The preprocessor command #include is used to 
link header files to the program.
    The definition section defines all the symbolic constants using the #define preprocessor command.
    Some variables are used in more than one function. Such variables are declared outside of all the functions, i.e., the variables commonly used 
in all functions are declared above the main( ) function. This process is known as global variable declaration.
    Every C program must have one main() function section, because it is the starting point of execution.
    main() section contains two parts: the declaration part and the executable part. The declaration part contains the declaration of variables 
which are used in the executable part.There is at least one statement in the executable part.
    These parts must appear in between the opening and closing braces of the main() function. All the statements in the declaration and executable 
parts end with a semicolon(;).
    The Sub-program section contains all the user-defined functions that are called in main() function. User-defined functions are generally 
placed immediately after the main() function.
    Remember that all the sections except the main() function section are optional., i.e.,they may or may not be needed.

/* A program to find the largest number */
#include <stdio.h> /* Link section */
void main() /* main() function */ 
{
	int a, b; //Declaration part
	printf("Enter two integer values : "); // Execution part
	// Read two values
	scanf("%d %d", &a, &b);
	(a>b)?printf("%d is larger than %d\n", a, b):
        printf("%d is larger than %d\n", b, a); 
}
/* End of main() function */
